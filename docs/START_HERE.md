# 🚀 START HERE

Welcome to the Academic AI Automation Kit. This is a professional tool to help you digest academic papers faster.

### How to use this product:
1. **For Non-Techies**: Use the **Google Colab Notebook**. It requires zero installation.
2. **For Local Users**: Install via `pip` and run `app.py` for a clean web interface.
3. **For Power Users**: Use the `main.py` CLI for fast batch processing.

### The Workflow:
`Paper Text` $\rightarrow$ `Automation Kit` $\rightarrow$ `Structured Summary` $\rightarrow$ `Notion/Zettelkasten`
