# 🛠️ INSTALLATION GUIDE

## Option A: Google Colab (Easiest)
1. Go to `notebooks/Academic_AI_Automation_Kit_Colab.ipynb`.
2. Upload it to Google Colab.
3. Run all cells.

## Option B: Local Installation
1. **Clone the repo**: `git clone https://github.com/pusakamediaid-dotcom/academic-ai-automation-kit.git`
2. **Install Python 3.9+**
3. **Install dependencies**: `pip install -r requirements.txt`
4. **(Optional) HF Token**: If you have a Hugging Face token, set it as an environment variable: `export HF_TOKEN='your_token_here'`. The tool works without it, but a token is more stable.
