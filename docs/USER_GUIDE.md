# 📖 USER GUIDE

## Choosing the Right Mode
- **Short Summary**: Best for deciding if a paper is worth reading fully.
- **Detailed Summary**: Best for understanding the "how" and "why" of a study.
- **Key Insights**: Best for extracting results, gaps, and conclusions.
- **Literature Review Notes**: Best for creating entries in a research database.

## Tips for Best Results
1. **Clean your text**: Remove headers, footers, and page numbers before pasting for better AI focus.
2. **Chunking**: The tool handles long texts, but if a paper is extremely complex, try processing section by section (e.g., Abstract $\rightarrow$ Methodology $\rightarrow$ Results).
3. **Verify**: Always double-check AI-generated citations.
