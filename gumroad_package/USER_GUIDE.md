# 📖 USER GUIDE (FOR BUYERS)

### Processing Modes Explained:
- **Short Summary**: The "Elevator Pitch" of the paper.
- **Detailed Summary**: The "Executive Summary".
- **Key Insights**: The "Bottom Line".
- **Literature Review Notes**: The "Database Entry".

### Pro Tip:
Paste the **Abstract** and the **Conclusion** together for the most accurate high-level summary.
